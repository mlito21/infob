Dispositivos de entrada

Concepto:

¿Qué son los dispositivos de entrada?


Los dispositivos de entrada, son aquellos que introducen en un sistema o computadora nueva información, es decir, que aumentan el contenido que esta puede recibir para agrandar la base de datos.



Teclado:

El teclado es uno de los principales dispositivos de entrada de información que utilizamos para interactuar e incluir datos en el computador. Normalmente el teclado se encuentra marcado por seis áreas, las cuales son: Teclas de función, teclas especiales, teclas de control, teclas para escribir o alfanuméricas, teclas de desplazamiento y el teclado numérico a la derecha del todo.

Mouse:

El Mouse o mas conocido como Ratón se trata de un dispositivo de entrada, el cual permite ingresar órdenes a la computadora, normalmente cuenta con dos botones: el izquierdo sirve para ejecutar acciones de activar, desplazar y acceder; el botón derecho se utiliza principalmente como atajo a menús, también cuentan una rueda o botón central que sirve para desplazar la pantalla hacia arriba o abajo.

El puntero del mouse puede tener distintas formas dependiendo de lo que estemos haciendo, las mas comunes son: 

Selección Normal: Dentro de las áreas de menús, sobre los botones y el escritorio.                          Ocupado: Cuando el sistema realiza una tarea importante y detiene el trabajo con el usuario.

Selección de texto: Dentro de áreas donde se puede escribir como un documento de Word o el Block de notas.

Trabajando en segundo plano: Cuando el sistema realiza otras tareas pero permite al usuario seguir utilizando el equipo.

Webcam:

Web Cam o Cámara digital, es un dispositivo de entrada que se conecta a una computadora, la cual puede capturar imagen y video, estos formatos son transmitirlas a través de Internet, ya sea a una página Web o a otra u otras computadoras de forma privada. 

Es un dispositivo que se conecta al puerto USB de la computadora así permite captar video y tomar fotos digital.

Micrófono:

Es un dispositivo de entrada de audio, el cual se pueden conectar a la computadora para grabar sonido o para comunicarse por internet con otras personas. Muchos equipos vienen con micrófonos incorporados dentro de la pantalla o monitor, especialmente los computadores portátiles.

Scanner:

Periférico de entrada, el cual se utiliza para "copiar", esto lo hace mediante el uso de la luz, teniendo como resultado en formato digital imágenes o documentos. Los escáneres normalmente permiten escanear un documento colocándolo de cara al panel de vidrio. Éste tipo de escáner es el más común .

Touchpad:

Un touchpad o "trackpad" es una superficie de control plana utilizada para mover el cursor y realizar otras funciones en una computadora. Comúnmente viene integrado en ordenadores portátiles y reemplazar la funcionalidad de un ratón, pero también existen para computador de mesa, el cual tiene la misma función.

Pueden detectar la aceleración y el movimiento lineal, lo que permite tener un control refinado con movimientos lentos y mover rápidamente el cursor por la pantalla con un movimiento rápido.
