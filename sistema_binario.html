Sistema Binario

💭 ¿Qué es el sistema binario?

El sistema binario es la base fundamental dentro los diferentes circuitos internos del computador, siendo el lenguaje con el cual se interpretaran los diferentes datos que se le proporcionen, para lo cual mediante el uso de una numeración par de 0 y 1 los que a su vez reciben el nombre de bit, se llevaran a cabo los diferentes procesos requeridos.

Es así, que para la medición binaria de la diferente información que posee el ordenador se utiliza un factor multiplicador 1024, a diferencia de otros sistemas de medida que se basan en multiplicar por 1000.
